Detail C / 3.1 - Beam to column web connection
Source: SGP26-CSPS combined - 20260424_Internal comments.pdf, sheet SGP26-CSPS-EX-CD-01.

Units: millimetres.

Included geometry:
- Column: assumed EC2/ECS UC203x203x46.
- Beam: assumed G1 UB254x146x37, using the marked-up UB254x146x37 value on sheet SE-02.
- Connection plate: PL 6 mm THK, modelled as 200 mm high x 120 mm projection from Detail 3.1.
- Column reinforcement/stiffeners: PL 8 mm THK plates, modelled on the column web faces plus top/bottom stiffener plates.
- Bolts: M16, modelled as three bolt assemblies on a vertical line using the 50-100-50 / 200 dimension shown in Detail 3.1.

Assumptions:
- The detail crop does not give a unique column/beam mark, so the model uses representative project profiles.
- Bolt holes are represented by bolt assemblies, not boolean-cut holes, for cleaner OBJ interchange.
- The visible 120 mm dimension is used for PL 6 projection; the smaller 5/60/30 marks are retained as drawing context and not fully detailed as slotted holes.
